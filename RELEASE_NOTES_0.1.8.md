# Focus Flow 0.1.8

## 布局重做

- Header 压缩为 sticky 紧凑顶栏，一行包含品牌名、搜索框、统计数字、番茄钟、操作按钮。
- 搜索框提到顶栏，随时可达。
- Today 主线移到左侧占大面积（60%），快速录入移到右侧更紧凑（40%）。
- 标签过滤和工具栏合并到顶栏第二行，工具栏按钮缩小为紧凑样式。
- 番茄钟从浮动遮挡改为内联到顶栏。
- 统计数字从 5 个大卡片改为内联紧凑格式。

## 卡片交互修复

- "更多"面板改用 React Portal 渲染到 document.body，彻底解决被其他元素遮挡的问题。
- 面板位置跟随按钮，滚动和窗口缩放时自动重新定位。
- 卡片背景加优先级色调（高/中/低优先级分别带玫瑰/琥珀/青色 10-12% 透明度）。
- 按钮标签简化为"转 Today"、"转 Batch"、"转 Review"格式。
- 主行展示 5 个按钮（主操作 + 所有流转 + 专注），"更多"里只放低频操作。

## 清理

- 移除中英文切换功能（LocaleToggle、LocaleProvider、i18n.ts）。
- 移除旧的 Workbench 组件，功能合并到顶栏工具栏。

## 验证

- `npm run lint`：通过，0 errors。
- `npm run build`：通过。
